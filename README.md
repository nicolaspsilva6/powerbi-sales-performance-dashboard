# 📊 Sales Performance Dashboard | Power BI

Dashboard interativo desenvolvido em **Power BI** com o objetivo de analisar o desempenho de vendas e fornecer **insights estratégicos para tomada de decisão**.

Este projeto demonstra habilidades em **modelagem de dados, transformação de dados e criação de visualizações analíticas** aplicadas a um cenário de vendas.



# 🎯 Objetivo do Projeto

O objetivo deste dashboard é permitir uma visão clara e estratégica da performance de vendas através de indicadores chave de negócio (KPIs), possibilitando:

- Monitoramento da **receita total**
- Análise da **performance por produto**
- Comparação de **vendas por região**
- Acompanhamento da **evolução temporal das vendas**
- Identificação de **tendências e oportunidades de crescimento**

# 🧠 Modelagem de Dados

O modelo de dados foi estruturado para garantir **performance e escalabilidade nas análises**, seguindo boas práticas de **Business Intelligence**.

Principais etapas:

- Importação e tratamento dos dados
- Transformações utilizando **Power Query**
- Estruturação do modelo relacional
- Criação de **medidas utilizando DAX**
- Desenvolvimento de visualizações analíticas



# 📊 Principais Indicadores (KPIs)

O dashboard apresenta indicadores importantes para análise de performance:

- Receita Total
- Volume de Vendas
- Ticket Médio
- Vendas por Região
- Vendas por Produto
- Crescimento ao longo do tempo
